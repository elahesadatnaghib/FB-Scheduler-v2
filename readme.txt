# Feature-base telescope scheduler

To schedule a number of nights, please use “run.py”.

To see the visualization of the scheduled nights use “Graphics.py”, please choose the date at the bottom of the file. 

To train the controller, please use “Training.py”.

Required data: 
1. slewMatrix.dat at NightDataInLIS/Constants/
2. fieldID.lis    at NightDataInLIS/Constants/
